Detailed Description

Provides access to shared asset content.

Holds track of how long current object is the last reference to shared data

void unicorn::utility::asset::Handler::Subscribe ( )
private

Subscribes to shared data.

Increments m_pRefCounter and checks its value. If it became 2, resets m_pLastReferenceTimestamp.

void unicorn::utility::asset::Handler::Unsubscribe ( )
private

Unsubscribes from shared data.

Decrements m_pRefCounter and checks its value:

  • if it is 0, deletes shared data
  • if it is 1, stores current timestamp to m_pLastReferenceTimestamp

Resets shared data pointers
