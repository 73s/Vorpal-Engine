Detailed Description

template<class Key, class T, class Hash = std::hash<Key>, class KeyEqual = std::equal_to<Key>, class Allocator = std::allocator<std::pair<const Key, T>>>
class vp::utility::concurrent::UnorderedMap< Key, T, Hash, KeyEqual, Allocator >

Simple mutex-protected std::unordered_map for concurrent access.

In future it would make sense to use some high performance concurrent hash map. For example from the Junction library.
